# ES6 Classes
A class contains Constructors and Functions.A Class is also called as blueprint of objects. In ES6 a class is created using __class__ keyword.
\
__Syntax:__
&nbsp;&nbsp;&nbsp;class Class_name {  
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;}

__Class Expression :__
&nbsp;&nbsp;&nbsp;var var_name = new Class_name {  
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;}
\
The ES6 Class supports inheritance and there are two types of classes i.e. super class(or parent class) and sub class(or child class)
